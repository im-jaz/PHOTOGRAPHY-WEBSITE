# LensStudio Photography Portfolio

## Description

LensStudio is a responsive multi-page photography portfolio website designed to showcase photography work through a modern and visually appealing interface. The website features a clean black and red theme, immersive imagery, and an easy-to-navigate layout that highlights photography services, featured collections, and creative storytelling.

## Features

- Full-screen hero section with a background image
- Responsive navigation bar
- About page introducing the photographer
- Gallery page showcasing photography collections
- Services page highlighting photography services
- Blog page with photography tips and articles
- Contact page with a contact form and FAQs
- Responsive design for desktop, tablet, and mobile devices
- Interactive hover effects and smooth transitions

## Technologies Used

- HTML5
- CSS3

## Project Setup

1. Clone the repository:

```bash
git clone git@github.com:im-jaz/PHOTOGRAPHY-WEBSITE.git
```

2. Open the project folder.

3. Open `index.html` in your web browser.

No additional software or dependencies are required since this project is built using HTML and CSS.
